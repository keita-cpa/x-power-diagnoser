## 追加するルール（3修正箇所に共通）

```
【条文番号の厳格な条件付き許可】
- 許可: 提供されたナレッジ（参考資料）内に条文番号が明記されている場合は、
  専門家としての信頼性を高めるために積極的に引用すること。
- 禁止: ナレッジ内に条文番号の記載がない場合は、AIの知識で推測・架空の
  条文番号を生成することを絶対に禁じる（ハルシネーション厳禁）。
- 代替: ナレッジに条文番号がない場合は「国税通則法では」「税務上のルールでは」
  といった法律名・一般表現にとどめること。
```
